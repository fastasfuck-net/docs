📦 Installation

NPM Installation

Installieren Sie unser Paket mit npm:

npm install devdocs-api --save

CDN Einbindung

Oder nutzen Sie unser CDN für eine schnelle Integration:

<script src="https://cdn.devdocs.io/v1/devdocs.min.js"></script>

🚀 Schnellstart

Erste Schritte

Hier ist ein einfaches Beispiel, um sofort loszulegen:

// Initialisierung der DevDocs API const api = new DevDocsAPI({ apiKey: 'ihr-api-schluessel', baseURL: 'https://api.devdocs.io/v1' }); // Erste API-Anfrage async function getData() { try { const response = await api.get('/users'); console.log('Benutzer:', response.data); } catch (error) { console.error('Fehler:', error.message); } } getData();

🔗 API Übersicht

Verfügbare Endpunkte

Unsere API bietet folgende Hauptendpunkte:

GET /users - Alle Benutzer abrufen
POST /users - Neuen Benutzer erstellen
GET /users/{id} - Spezifischen Benutzer abrufen
PUT /users/{id} - Benutzer aktualisieren
DELETE /users/{id} - Benutzer löschen

Alle Endpunkte unterstützen JSON-Format und erfordern eine gültige API-Authentifizierung.
