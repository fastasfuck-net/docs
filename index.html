API Reference

Simple, powerful API designed for maximum performance

Core Methods

fast.optimize(fn, options?)

Optimizes any function for maximum performance

fast.optimize(arr => arr.sort())

fast.cache(fn, options?)

Adds intelligent result caching to any function

fast.cache(expensiveFn, { maxSize: 100 })

fast.parallel(fn, options?)

Distributes work across multiple CPU cores

fast.parallel(heavyComputation, { workers: 4 })

Advanced Methods

fast.auto(fn, options?)

Machine learning-based auto-optimization

fast.auto(fn, { learnFromUsage: true })

fast.batch(fns[], options?)

Optimizes multiple functions together

fast.batch([fn1, fn2, fn3])

fast.profile(fn)

Detailed performance profiling and analysis

const stats = fast.profile(myFunction)
